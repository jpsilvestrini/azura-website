# Swagger / OpenAPI 3.0

AzuraJS includes full support for API documentation using Swagger/OpenAPI 3.0, featuring a modern dark mode interface and integrated "Try It Out" functionality.

> **Note:** The AzuraJS Swagger system automatically generates OpenAPI 3.0 documentation from your controllers and decorators, without manual configuration.

## Installation

Swagger is already included in AzuraJS. Just import it from the `azurajs/swagger` submodule:

```typescript
import { setupSwaggerWithControllers } from "azurajs/swagger";

```

## Quick Setup

The simplest way to add Swagger to your application is with a single line:

```typescript
import { AzuraClient } from "azurajs";
import { setupSwaggerWithControllers } from "azurajs/swagger";
import { Controller, Get, Post, Req, Res } from "azurajs/decorators";

@Controller("/users")
class UserController {
  private users = [
    { id: "1", name: "Alice", email: "alice@example.com" },
    { id: "2", name: "Bob", email: "bob@example.com" },
  ];

  @Get("/")
  async listUsers(@Req() req, @Res() res) {
    const page = Number(req.query.page) || 1;
    const limit = Number(req.query.limit) || 10;
    const start = (page - 1) * limit;
    const items = this.users.slice(start, start + limit);
    res.json({ items, total: this.users.length, page });
  }

  @Get("/:id")
  async getUser(@Req() req, @Res() res) {
    const user = this.users.find((u) => u.id === req.params.id);
    if (!user) return res.status(404).json({ error: "User not found" });
    res.json(user);
  }

  @Post("/")
  async createUser(@Req() req, @Res() res) {
    const { name, email } = req.body;
    if (!name || !email) return res.status(400).json({ error: "name and email are required" });
    const user = { id: String(Date.now()), name, email };
    this.users.push(user);
    res.status(201).json(user);
  }
}

const app = new AzuraClient();

setupSwaggerWithControllers(
  app,
  {
    title: "My API",
    description: "API documentation",
    version: "1.0.0",
  },
  [UserController]
);

await app.listen();

```

> **Access the documentation at:**
> * **Interactive UI:** `http://localhost:3000/docs`
> * **OpenAPI JSON:** `http://localhost:3000/api-spec.json`

## Documentation Decorators

### @Swagger() - New Simplified API ‚≠ê

The `@Swagger` decorator unifies **all** documentation in a single place, making it much easier and more intuitive to document your APIs:

```typescript
import { AzuraClient } from "azurajs";
import { Controller, Get, Post, Req, Res } from "azurajs/decorators";
import { Swagger, setupSwaggerWithControllers } from "azurajs/swagger";

@Controller("/products")
class ProductController {
  private products = [
    { id: "p1", name: "Laptop", price: 1299.99 },
    { id: "p2", name: "Mouse", price: 29.99 },
  ];

  @Get("/")
  @Swagger({
    summary: "List products",
    description: "Returns a paginated list of products",
    tags: ["Products"],
    parameters: [
      { name: "page", in: "query", schema: { type: "number" }, example: 1 },
      { name: "limit", in: "query", schema: { type: "number" }, example: 10 },
    ],
    responses: {
      200: {
        description: "List of products",
        example: [
          { id: "p1", name: "Laptop", price: 1299.99 },
          { id: "p2", name: "Mouse", price: 29.99 },
        ],
      },
    },
  })
  async list(@Req() req, @Res() res) {
    const page = Number(req.query.page) || 1;
    const limit = Number(req.query.limit) || 10;
    const items = this.products.slice((page - 1) * limit, page * limit);
    res.json(items);
  }

  @Get("/:id")
  @Swagger({
    summary: "Get product by ID",
    description: "Returns detailed information about a product",
    tags: ["Products"],
    parameters: [
      {
        name: "id",
        in: "path",
        description: "Product ID",
        required: true,
        schema: { type: "string" },
        example: "p1",
      },
    ],
    responses: {
      200: {
        description: "Product found",
        example: { id: "p1", name: "Laptop", price: 1299.99 },
      },
      404: {
        description: "Product not found",
        example: { error: "Product not found" },
      },
    },
  })
  async get(@Req() req, @Res() res) {
    const product = this.products.find((p) => p.id === req.params.id);
    if (!product) return res.status(404).json({ error: "Product not found" });
    res.json(product);
  }

  @Post("/")
  @Swagger({
    summary: "Create new product",
    tags: ["Products"],
    requestBody: {
      description: "Product data",
      required: true,
      content: {
        "application/json": {
          schema: {
            type: "object",
            properties: {
              name: { type: "string" },
              price: { type: "number" },
            },
            required: ["name", "price"],
          },
          example: { name: "Keyboard", price: 79.99 },
        },
      },
    },
    responses: {
      201: {
        description: "Product created",
        example: { id: "p3", name: "Keyboard", price: 79.99 },
      },
      400: {
        description: "Invalid data",
        example: { error: "Invalid data" },
      },
    },
  })
  async create(@Req() req, @Res() res) {
    const { name, price } = req.body;
    if (!name || typeof price !== "number") return res.status(400).json({ error: "Invalid data" });
    const newProduct = { id: `p${Date.now()}`, name, price };
    this.products.push(newProduct);
    res.status(201).json(newProduct);
  }
}

const app = new AzuraClient();

setupSwaggerWithControllers(
  app,
  {
    title: "Products API",
    description: "Automatic documentation with decorators",
    version: "1.0.0",
    tags: [{ name: "Products", description: "Product management" }],
  },
  [ProductController]
);

await app.listen();

```

**Advantages of @Swagger:**

* ‚úÖ **All in one place** - No need to use multiple decorators
* ‚úÖ **More readable** - Grouped and organized configuration
* ‚úÖ **Less code** - A single declaration instead of 4-5 decorators
* ‚úÖ **TypeScript** - Autocomplete and type validation
* ‚úÖ **Integrated examples** - Define examples directly in the responses

## Simple JavaScript (Without Decorators)

If you prefer pure JavaScript or don't want to use decorators, use the `addDoc()` method:

```javascript
import { AzuraClient } from "azurajs";
import { setupSwagger, SwaggerGenerator } from "azurajs/swagger";

const app = new AzuraClient();
const swagger = new SwaggerGenerator({
  title: "Simple API",
  description: "API documented simply",
  version: "1.0.0",
  tags: [
    { name: "Users", description: "User management" },
    { name: "Utils", description: "Utilities" },
  ],
});

const users = [
  { id: "1", name: "Mariana", email: "mariana@example.com" },
  { id: "2", name: "Carlos", email: "carlos@example.com" },
];

app.get("/users", (req, res) => {
  res.json(users);
});

swagger.addDoc({
  method: "GET",
  path: "/users",
  summary: "List users",
  description: "Returns all registered users",
  tags: ["Users"],
  responses: {
    200: {
      description: "List of users",
      example: users,
    },
  },
});

app.get("/utils/encode", (req, res) => {
  const text = req.query.q || "";
  const base64 = Buffer.from(String(text)).toString("base64");
  res.json({ input: text, output: base64 });
});

swagger.addDoc({
  method: "GET",
  path: "/utils/encode",
  summary: "Base64 encode",
  description: "Converts text to Base64",
  tags: ["Utils"],
  parameters: [
    {
      name: "q",
      in: "query",
      description: "Text to be encoded",
      required: true,
      schema: { type: "string" },
      example: "hello",
    },
  ],
  responses: {
    200: { description: "Encoded text", example: { input: "hello", output: "aGVsbG8=" } },
    400: { description: "Missing parameter", example: { error: "Query 'q' is required" } },
  },
});

app.post("/users", (req, res) => {
  const newUser = { id: String(Date.now()), ...req.body };
  users.push(newUser);
  res.status(201).json(newUser);
});

swagger.addDoc({
  method: "POST",
  path: "/users",
  summary: "Create new user",
  tags: ["Users"],
  requestBody: {
    description: "User data",
    required: true,
    content: {
      "application/json": {
        schema: {
          type: "object",
          properties: { name: { type: "string" }, email: { type: "string" } },
          required: ["name", "email"],
        },
        examples: { default: { value: { name: "John Doe", email: "john@example.com" } } },
      },
    },
  },
  responses: {
    201: {
      description: "User created",
      example: { id: "123", name: "John Doe", email: "john@example.com" },
    },
    400: { description: "Invalid data", example: { error: "Invalid data" } },
  },
});

setupSwagger(app, swagger);
await app.listen();
console.log("üöÄ Server: http://localhost:3000");
console.log("üìö Docs: http://localhost:3000/docs");

```

**Advantages of `addDoc()`:**

* ‚úÖ **Pure JavaScript** - Works without TypeScript or decorators
* ‚úÖ **Simple** - Less verbose than SwaggerHelpers.createResponse
* ‚úÖ **Automatic** - Generates OpenAPI responses from examples
* ‚úÖ **Flexible** - Mix with traditional AzuraJS routes

## Complete Example (TypeScript with Decorators)

```typescript
import { Controller, Get, Post, Delete, Req, Res } from "azurajs/decorators";
import { Swagger } from "azurajs/swagger";

@Controller("/todos")
class TodoController {
  private todos = [
    { id: "t1", title: "Buy milk", completed: false },
    { id: "t2", title: "Send report", completed: true },
  ];

  @Get("/")
  @Swagger({
    summary: "List all tasks",
    tags: ["Todos"],
    parameters: [{ name: "completed", in: "query", schema: { type: "boolean" }, example: false }],
    responses: {
      200: {
        description: "List of tasks",
        example: [
          { id: "t1", title: "Buy milk", completed: false },
          { id: "t2", title: "Send report", completed: true },
        ],
      },
    },
  })
  async list(@Req() req, @Res() res) {
    const completed = req.query.completed;
    if (typeof completed !== "undefined") {
      const filtered = this.todos.filter((t) => String(t.completed) === String(completed));
      return res.json(filtered);
    }
    res.json(this.todos);
  }

  @Get("/:id")
  @Swagger({
    summary: "Get task by ID",
    tags: ["Todos"],
    parameters: [
      { name: "id", in: "path", required: true, schema: { type: "string" }, example: "t1" },
    ],
    responses: {
      200: {
        description: "Task found",
        example: { id: "t1", title: "Buy milk", completed: false },
      },
      404: { description: "Task not found", example: { error: "Not found" } },
    },
  })
  async get(@Req() req, @Res() res) {
    const todo = this.todos.find((t) => t.id === req.params.id);
    if (!todo) return res.status(404).json({ error: "Not found" });
    res.json(todo);
  }

  @Post("/")
  @Swagger({
    summary: "Create new task",
    tags: ["Todos"],
    requestBody: {
      description: "Task data",
      required: true,
      content: {
        "application/json": {
          schema: {
            type: "object",
            properties: { title: { type: "string" }, completed: { type: "boolean" } },
            required: ["title"],
          },
          example: { title: "Study AzuraJS", completed: false },
        },
      },
    },
    responses: {
      201: {
        description: "Task created",
        example: { id: "t3", title: "Study AzuraJS", completed: false },
      },
      400: { description: "Invalid data", example: { error: "Invalid data" } },
    },
  })
  async create(@Req() req, @Res() res) {
    const { title, completed = false } = req.body;
    if (!title) return res.status(400).json({ error: "title is required" });
    const newTodo = { id: `t${Date.now()}`, title, completed };
    this.todos.push(newTodo);
    res.status(201).json(newTodo);
  }

  @Delete("/:id")
  @Swagger({
    summary: "Delete task",
    tags: ["Todos"],
    parameters: [
      { name: "id", in: "path", required: true, schema: { type: "string" }, example: "t1" },
    ],
    responses: {
      204: { description: "Task deleted" },
      404: { description: "Task not found", example: { error: "Not found" } },
    },
  })
  async remove(@Req() req, @Res() res) {
    const index = this.todos.findIndex((t) => t.id === req.params.id);
    if (index === -1) return res.status(404).json({ error: "Not found" });
    this.todos.splice(index, 1);
    res.status(204).send();
  }
}

```

## Individual Decorators - Complete Example

If you prefer using the traditional decorators separately:

```typescript
import { Controller, Get, Post, Put, Delete, Req, Res } from "azurajs/decorators";
import { ApiDoc, ApiResponse, ApiParameter, ApiBody, ApiTags } from "azurajs/swagger";

@Controller("/customers")
@ApiTags("Customers")
class CustomersController {
  private customers = [{ id: "c1", name: "Company A", contact: "contact@company.com" }];

  @Get("/")
  @ApiDoc({ summary: "List all customers", description: "Retrieve a paginated list of customers" })
  @ApiParameter("page", "query", {
    description: "Page number",
    schema: { type: "integer", default: 1 },
  })
  @ApiParameter("limit", "query", {
    description: "Items per page",
    schema: { type: "integer", default: 10 },
  })
  @ApiResponse(200, "Success", {
    example: { customers: [{ id: "c1", name: "Company A" }], total: 1, page: 1 },
  })
  async list(@Req() req, @Res() res) {
    res.json({
      customers: this.customers,
      total: this.customers.length,
      page: req.query.page || 1,
    });
  }

  @Get("/:id")
  @ApiDoc({ summary: "Get customer by ID", description: "Retrieve detailed customer information" })
  @ApiParameter("id", "path", {
    description: "Customer ID",
    required: true,
    schema: { type: "string" },
  })
  @ApiResponse(200, "Customer found", {
    example: { id: "c1", name: "Company A", contact: "contact@company.com" },
  })
  @ApiResponse(404, "Customer not found")
  async get(@Req() req, @Res() res) {
    const customer = this.customers.find((c) => c.id === req.params.id);
    if (!customer) return res.status(404).json({ error: "Customer not found" });
    res.json(customer);
  }

  @Post("/")
  @ApiDoc({ summary: "Create new customer", description: "Create a new customer in the system" })
  @ApiBody({
    description: "Customer data",
    required: true,
    example: { name: "Company B", contact: "b@company.com" },
  })
  @ApiResponse(201, "Customer created", {
    example: { id: "c2", name: "Company B", contact: "b@company.com" },
  })
  @ApiResponse(400, "Invalid data")
  async create(@Req() req, @Res() res) {
    const { name, contact } = req.body;
    if (!name || !contact) return res.status(400).json({ error: "Invalid data" });
    const newCustomer = { id: `c${Date.now()}`, name, contact };
    this.customers.push(newCustomer);
    res.status(201).json(newCustomer);
  }

  @Put("/:id")
  @ApiDoc({ summary: "Update customer" })
  @ApiParameter("id", "path", {
    description: "Customer ID",
    required: true,
    schema: { type: "string" },
  })
  @ApiBody({ description: "Updated customer data", example: { name: "Company A Updated" } })
  @ApiResponse(200, "Customer updated")
  @ApiResponse(404, "Customer not found")
  async update(@Req() req, @Res() res) {
    const customer = this.customers.find((c) => c.id === req.params.id);
    if (!customer) return res.status(404).json({ error: "Customer not found" });
    Object.assign(customer, req.body);
    res.json(customer);
  }

  @Delete("/:id")
  @ApiDoc({ summary: "Delete customer" })
  @ApiParameter("id", "path", { description: "Customer ID", required: true })
  @ApiResponse(204, "Customer deleted")
  @ApiResponse(404, "Customer not found")
  async remove(@Req() req, @Res() res) {
    const idx = this.customers.findIndex((c) => c.id === req.params.id);
    if (idx === -1) return res.status(404).json({ error: "Customer not found" });
    this.customers.splice(idx, 1);
    res.status(204).send();
  }
}

```

## Manual Setup (Advanced)

If you need more control, you can configure it manually:

```typescript
import { AzuraClient } from "azurajs";
import { SwaggerIntegration } from "azurajs/swagger";
import { SwaggerGenerator } from "azurajs/swagger";
import { getControllerMetadata } from "azurajs/decorators";
import UserController from "./controllers/UserController";

const app = new AzuraClient();

const generator = new SwaggerGenerator({
  title: "My API",
  version: "1.0.0",
  description: "Manual Swagger generator",
});

const instance = new UserController();
const metadata = getControllerMetadata(UserController);
if (metadata && metadata.routes && metadata.routes.length) {
  generator.addController(instance, metadata.routes, metadata.prefix);
}

const integration = new SwaggerIntegration(app, {
  title: "My API",
  version: "1.0.0",
  path: "/docs",
});

integration.setup();
app.registerController(UserController);

await app.listen();

```

## Best Practices

### 1. Use @ApiTags for Organization

Group related endpoints with tags:

```typescript
@Controller("/users")
@ApiTags("Users", "Management")
class UserController {}

@Controller("/products")
@ApiTags("Products", "E-commerce")
class ProductController {}

```

### 2. Provide Detailed Examples

Examples help developers understand the expected format:

```typescript
@ApiBody({
  description: "User registration data",
  example: {
    name: "John Doe",
    email: "john@example.com",
    password: "secure123",
    preferences: {
      newsletter: true,
      notifications: ["email", "push"]
    }
  }
})

```

### 3. Document All Status Codes

Include all possible responses:

```typescript
@ApiResponse(200, "Success")
@ApiResponse(400, "Bad request")
@ApiResponse(401, "Unauthorized")
@ApiResponse(403, "Forbidden")
@ApiResponse(404, "Not found")
@ApiResponse(500, "Server error")

```

### 4. Use Clear Descriptions

Be descriptive in summaries and descriptions:

```typescript
@ApiDoc({
  summary: "Update user profile",
  description: "Updates the authenticated user's profile information. Only provided fields will be updated. Requires valid JWT token."
})

```

### 5. Mark Deprecated Endpoints

Inform when endpoints are deprecated:

```typescript
@Get("/old-api")
@ApiDeprecated()
@ApiDoc({
```

## Troubleshooting

### Documentation does not appear

Ensure that:

1. You imported the decorators from `azurajs/swagger`
2. The decorators are applied to the methods
3. The controllers were passed to `setupSwaggerWithControllers()`

### Try It Out does not work

Check:

1. CORS is configured correctly
2. The server is running at the expected URL
3. The parameters are being filled in correctly

### TypeScript Types do not work

Ensure you import the types:

```typescript
import type { SwaggerConfig } from "azurajs/swagger";

```
